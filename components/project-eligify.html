Eligify: Secure Exam Eligibility Verification

Group Project | Web Application Development

The Challenge

For many students, answering the simple question "Am I actually eligible for this exam?" is surprisingly complex. Our goal was to solve this confusion, but we didn't just want a basic prototype. The real challenge lay in building a system capable of handling sensitive personal data and verifying scanned documents within a secure, production-ready environment.

Our Solution

We engineered a platform that cross-references student data against exam rules to provide accurate results. The system automates document verification, eliminates stored passwords for better privacy, and provides real-time feedback, all while maintaining a hardened security posture against common web vulnerabilities.

Work Distribution & Architecture

  • Secure Backend Architecture: MVC architecture with Flask Blueprints. Focus on hardening against XSS/SQLi, rate limiting, and secure file validation.
  • Hybrid Document Parsing: A hybrid engine that reads digital PDFs directly and automates switching to OCR (Tesseract) for scanned images.
  • Identity & Data Integrity: Google OAuth 2.0 implementation and normalized SQLAlchemy schema for audit trails.
  • Real-Time Feedback: Client-side MVC with ES6 modules for instant eligibility validation.

Outcome & Reflection

This project was a comprehensive exercise in full-stack security and logic integration. By combining robust backend protection with intelligent parsing and a user-friendly interface, we created a tool that makes the complex simple.

Tech Stack: Python, Flask, Tesseract OCR, Google Identity Services, Tailwind CSS
